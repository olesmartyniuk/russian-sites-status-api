Про сайт

Даний сервіс моніторить російські та білоруські сайти та перевіряє їх доступність. Перевірка здійснюється з різних куточків світу по протоколу HTTP з частотою раз на 30 сек. Один з серверів моніторингу знаходиться на території мордору. Серед них сайти банків, держ. установ, ЗМІ.

Список сайтів береться з дружнього ресурсу Допомога в DDOS атаках.

Доступний API для автоматизації. Сайт та API доступні всім без авторизації.

Сервіс створено за підтримки Lviv .NET Community.

About site

This service monitors Russian and Belarusian sites and checks their availability. The checks are performed from different parts of the world via HTTP with a frequency of 30 seconds. One of the monitoring servers is located in the territory of the Russian Federation. There are banks, media, government sites, and others among them.

The list of sites is taken from our friends Help in DDOS attacks.

API for automation is available here. The site and the API are available to anyone without authorization.

Creation of service was supported by Lviv .NET Community.
